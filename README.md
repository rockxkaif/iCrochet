# ğŸ§µ About

iCrochet is a React-based portfolio designed to display crochet products, gallery images, and basic business information.  
The site includes:

- **Home** (Hero + About + Instagram CTA + Contact form)  
- **Products** (Grid of product cards)  
- **About** (Info sections + Gallery)  
- **Contact Form**

The design is simple, aesthetic, and crafted for small businesses selling handmade products.
